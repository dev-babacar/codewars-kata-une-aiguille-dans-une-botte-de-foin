# codewars-kata-une-aiguille-dans-une-botte-de-foin
Entraînement Kata : Une aiguille dans une botte de foin
Saurez-vous trouver l'aiguille dans la botte de foin ?

Écrivez une fonction findNeedle()qui prend un arraytableau plein de données inutiles mais contenant un seul élément."needle"

Une fois que votre fonction a trouvé l'aiguille, elle doit renvoyer un message (sous forme de chaîne de caractères) indiquant :

"found the needle at position "de plus, indexil a trouvé l'aiguille, donc :
Exemple (Entrée --> Sortie)
["hay", "junk", "hay", "hay", "moreJunk", "needle", "randomJunk"] --> "found the needle at position 5"
